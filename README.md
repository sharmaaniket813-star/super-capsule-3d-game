# 🎮 Super Capsule — 3D Platformer Game
> Independent Project (2025–2026)

A fully playable 3D platformer game built from scratch in Unity, 
featuring custom physics, multi-level design, animated sequences, 
and a complete HUD system.

---

## 🎯 Features

- 🕹️ Full player movement, jumping, and collision physics
- 🏆 Item collection system with score tracking
- 👾 Waypoint-based moving platforms and obstacles
- 🖥️ Complete HUD — health, score, and timer
- 🎬 Animated sequences using Unity Timeline
- 🗺️ Multiple levels with Start, Game, and End screens

---

## 🛠️ Tech Stack

| Tool | Usage |
|------|-------|
| Unity 2020.3 | Game engine |
| C# | All game logic (11 custom scripts) |
| Unity Timeline | Cutscenes and animations |
| TextMeshPro | UI and HUD text rendering |

---

## 📜 Custom Scripts

| Script | Purpose |
|--------|---------|
| `PlayerMovement.cs` | Player controls and physics |
| `PlayerLife.cs` | Health and death handling |
| `GameManager.cs` | Game state and score tracking |
| `HUDManager.cs` | On-screen UI updates |
| `ItemCollector.cs` | Collectible item logic |
| `WaypointFollower.cs` | Moving platform/enemy paths |
| `StickyPlatform.cs` | Platform attachment physics |
| `StartMenu.cs` | Main menu interactions |
| `EndMenu.cs` | End screen and restart logic |
| `Finish.cs` | Level completion trigger |
| `Rotate.cs` | Object rotation animations |
